# 主题

开发主题功能，可以自行配置主题颜色等。



```json
{
    "name": "主题名称",
    "light": {
        "colors": {
            "highlight": "#D3D3D3",
            "text": "#D3D3D3",
            "subText": "#D3D3D3",
            "background": "#D3D3D3",
            "subBackground": "#D3D3D3"
        }
    },
    "dark": {
        "colors": {
            "highlight": "#D3D3D3",
            "text": "#D3D3D3",
            "subText": "#D3D3D3",
            "background": "#D3D3D3",
            "subBackground": "#D3D3D3"
        }
    }
}
```
